# 社内向け 問題作成・回答ツール（静的ファイル版）

Node.js / Python / サーバー不要で、HTMLファイルを直接開くだけで使える単一選択問題ツールです。

## ファイル構成

- `index.html` : 回答者向けトップ（受験者情報入力）
- `exam.html` : 回答画面
- `result.html` : 回答結果画面（CSV出力）
- `admin.html` : 作成者向け管理画面（簡易パスワードあり）
- `style.css` : 共通スタイル
- `questions.js` : 初期問題データ（埋め込み方式）
- `app.js` : 回答者向け処理
- `admin.js` : 作成者向け処理
- `README.md` : 本手順

## 使い方

1. このフォルダを任意の場所に保存します。
2. `index.html` をダブルクリックしてブラウザで開きます。
3. 回答者は「所属部署 / 社員番号 / 名前」を入力して受験します。
4. 回答後に `result.html` で採点結果を確認し、CSVをダウンロードできます。
5. 作成者は `admin.html` を開き、パスワードを入力して問題管理を行います。

## 作成者画面

- 初期パスワード: `admin123`
- できること:
  - 問題作成 / 編集 / 削除
  - 問題データのエクスポート（JSON/JS）
  - 問題データのインポート（JSON または `window.DEFAULT_QUESTIONS = ...` 形式）

## データ保存

- 問題データ: ブラウザの `localStorage` に保存
- 回答結果: 回答完了時に画面表示し、CSVとしてダウンロード可能

## ローカル実行時の互換性について

- `file://` で開く環境では `fetch("questions.json")` が制限される場合があります。
- そのため本ツールは `questions.js` の埋め込みデータ方式を標準採用しています。

## セキュリティ上の制約（重要）

このツールはサーバーを使用しない静的構成です。したがって、以下の制約があります。

- 厳密な認証・認可はできません。
- 作成者画面のパスワードは「簡易保護」です（ブラウザ上での制御）。
- 機密性の高い用途では、サーバー型システムへの移行を検討してください。

## 補足

- スマホ最適化はしていません（PCブラウザ前提）。
- データは同一ブラウザ内に保存されるため、ブラウザデータ削除時に消えることがあります。
